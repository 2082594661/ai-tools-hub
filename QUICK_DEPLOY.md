# 🚀 一键部署指南

## 你现在只需运行一个命令:

```bash
cd C:\Users\龙潜\seo-website
python deploy_to_github.py
```

## 脚本会自动:

1. ✅ 创建 GitHub 仓库
2. ✅ 推送 20 篇文章
3. ✅ 启用 GitHub Pages
4. ✅ 返回网站地址

## 你需要准备:

1. **GitHub 用户名**
2. **GitHub Token** (下面教你怎么获取)

## 如何获取 GitHub Token:

1. 打开: https://github.com/settings/tokens
2. 点击: **Generate new token (classic)**
3. 勾选: **repo** (第一个大选项)
4. 点击: **Generate token**
5. **复制**生成的 token (只显示一次!)

## 运行后:

脚本会问你要用户名和 token，输入后自动完成所有步骤。

5 分钟后，你的网站会在这里:
```
https://你的用户名.github.io/ai-tools-hub/
```
